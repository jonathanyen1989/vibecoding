# FocusLens Product Positioning

## English

FocusLens is a real-time spotlight and zoom tool for live presentations, live demos, and live meetings.

FocusLens is not a post-production zoom tool for recorded videos. It is also not a generic screen magnifier. It is designed for presenters who need to guide audience attention during a live screen-sharing or projection session.

During a presentation, a remote meeting, a product walkthrough, or an on-site demo, presenters often need to quickly enlarge one important region of the screen without interrupting the flow of the talk. FocusLens helps them select a region, zoom it, and add a professional visual focus effect using a mask, border, feathering, blur, and watermark.

The product goal is to fill the gap between video-editing zoom tools and system magnifiers. FocusLens is built specifically for live explanation, where speed, clarity, and visual polish matter.

## 中文

FocusLens 是一款面向 live presentation、live demo 和 live meeting 的实时聚焦放大工具。

FocusLens 不是录屏后的后期放大工具，也不是普通屏幕放大镜。它专门服务于现场演示、远程会议共享和产品 demo 中的实时讲解场景，帮助演示者在讲解过程中快速引导观众注意力。

在真实 presentation、远程会议共享、教学讲解和现场 demo 中，演示者经常需要临时、快速、优雅地放大屏幕上的某个关键区域，同时不打断当前讲解节奏。FocusLens 可以帮助演示者快速框选并放大关键区域，并通过蒙版、边框、羽化、背景虚化和水印形成更专业的视觉聚焦效果。

FocusLens 的目标是补上录屏后期放大工具和系统放大镜之间的空白。它不是后期工具，也不是系统级通用缩放功能，而是一个为现场讲解设计的实时聚焦放大工具。
