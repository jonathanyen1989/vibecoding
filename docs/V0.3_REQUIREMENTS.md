# FocusLens macOS V0.3 Requirements

## Product Positioning

FocusLens is a macOS presentation utility for live talks, projector demos, product walkthroughs, training, and remote screen sharing. The tool helps the speaker quickly highlight a screen region without interrupting the talk.

## Success Criteria

- The speaker can trigger focus mode with a two-key shortcut during a live presentation.
- The overlay appears on the screen that contains the mouse.
- The selected region, zoomed screenshot, mask hole, and border stay aligned.
- Visual settings can be adjusted before a talk and persist between launches.
- The app can run as a menu bar utility without requiring an always-visible main window.

## In Scope

### Menu Bar Utility

- Show a FocusLens status item in the macOS menu bar.
- Provide menu actions:
  - Start Focus
  - Settings
  - Quit
- Keep the app running in the background after launch.

### Configurable Visuals

- Mask opacity:
  - Range: `20%` to `85%`
  - Default: `59%`
- Background blur:
  - Range: `0` to `12`
  - Default: `4`
- Feather strength:
  - Range: `0` to `100`
  - Default: `60`
- Border color:
  - Default: `#F6C945`
  - Editable with a native color picker.
- Max zoom:
  - Range: `1.2x` to `4.0x`
  - Default: `2.5x`

### Configurable Shortcut

- Default shortcut: `Control + W`.
- User can record a new shortcut in Settings.
- Shortcut must include at least one modifier key plus one regular key.
- The app re-registers the shortcut immediately after saving.

### Presentation-Friendly Defaults

- Keep static screenshot mode.
- Keep `Esc` as the fast exit action.
- Use faster animation than the Windows prototype:
  - Zoom in: `550ms`
  - Zoom out: `380ms`

### Icon Direction

- Create a polished icon for the app/menu bar identity.
- Visual direction:
  - dark presentation-tool feel
  - central focus window/lens
  - soft glow ring
  - professional macOS utility style

## Out of Scope

- Realtime screen streaming.
- Cross-screen selection.
- Cloud sync.
- User accounts.
- Complex installer.
- Automatic update system.
- Per-app shortcut profiles.

## Acceptance Tests

1. Launch the app and see a menu bar item.
2. Open Settings from the menu bar.
3. Change mask opacity, border color, feather strength, max zoom, and shortcut.
4. Close and relaunch; settings persist.
5. Trigger focus mode with the configured shortcut.
6. Drag a region; mask, border, and zoom appear with configured values.
7. Press `Esc`; overlay exits.
8. Quit from menu bar without crashing.
